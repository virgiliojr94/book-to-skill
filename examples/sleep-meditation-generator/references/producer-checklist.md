# Producer Checklist

Use this verbatim in the generated output, adjusted only for the chosen duration.

---

## Recording

- [ ] Voice recorded **dry, close, warm** — proximity effect is acceptable here, even encouraged
- [ ] No mouth clicks (drink water, use de-click software if needed)
- [ ] No sharp sibilance (de-essing in mix)
- [ ] Speaking rate **85–95 wpm** measured with stopwatch on a representative passage
- [ ] Pause markers respected — `[pause 4 sec]` is 4 real seconds, not "a breath"
- [ ] Voice in lower part of the speaker's natural range, never strained
- [ ] One full take preferred; if comp'ing, hide edits inside pauses

## Music bed

- [ ] No sudden changes, no bright rhythmic percussion, no dramatic climax
- [ ] Tempo ≤ 65 BPM or no clear pulse
- [ ] Frequency check: no harsh 2–5 kHz, soft low end (no rumble), nothing above ~10 kHz prominent
- [ ] No loop click at the seam — crossfade or stretch the loop edge
- [ ] Music starts ~3–5 sec before voice
- [ ] Music continues 30–60 sec after voice

## Mixing

- [ ] Voice ~6 dB above music bed during voice sections
- [ ] Soft auto-ducking, no obvious pumping
- [ ] Integrated loudness **−20 to −24 LUFS** (quieter than podcast standard)
- [ ] True peak **≤ −3 dBTP**
- [ ] No noise floor hiss, no hum, no abrupt ending
- [ ] Clean fade-out (15–30 sec)
- [ ] Stereo image: music wider than voice; voice centered, narrow

## Export

- [ ] WAV master: 24-bit / 48 kHz
- [ ] Distribution: MP3 192 kbps (streaming) and/or AAC 128 kbps
- [ ] Filename includes language code and duration (e.g. `sleep-meditation-en-4min-v1.mp3`)
- [ ] Test playback on:
  - [ ] Phone speaker (low volume)
  - [ ] Earbuds (in-bed scenario)
  - [ ] Bluetooth speaker
- [ ] Verify there are no unexpected loud peaks at any point

## Publication

- [ ] Title is calm: "Sleep Meditation", "Night Reset", "Deep Rest Audio" — never "Cure for Insomnia"
- [ ] Description includes safe-use note (see `safety-copy.md`)
- [ ] Description includes the clinical-referral line — always
- [ ] No subliminal claims, no Hz-healing claims, no medical promises
- [ ] Thumbnail/cover art: low-contrast, warm, no bright colors, no faces with intense expression

## A/B Testing

Always pair the production with at least one variant for testing. Suggested pairings:

| Variant A | Variant B | Test what |
|---|---|---|
| Voice + music bed | Voice only with light room tone | Does music help or distract? |
| Music carries 30s fade | Music carries 60s fade | Sleep-onset perception |
| Female voice | Male voice (or neutral) | Listener preference |
| Metaphor: warm room | Metaphor: quiet shore | Engagement / replay intention |
| 4 min track | 6 min track | Optimal duration for the audience |

## User Feedback to Collect

Use sleep-diary style questions (never diagnostic):

1. Did you listen all the way through? (yes / partial / stopped)
2. How relaxing was it? (1–5)
3. Did you fall asleep during the track? (yes / no / not sure)
4. Did anything in the audio create irritation or alertness? (open text)
5. Would you replay it tomorrow night? (yes / no)
6. Morning grogginess afterward? (none / some / a lot)

**Never ask**: "Did this cure your insomnia?" — that question violates principle 1.

## Iteration Discipline

- Remove any line that creates pressure, sadness, alertness, or expectation
- If listeners report being kept awake by a specific phrase, **cut it** — do not defend it
- Re-test after every change — at least 5 listeners before deciding
