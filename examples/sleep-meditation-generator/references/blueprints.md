# Timing Blueprints

Each blueprint specifies the section split for a target duration. Sections are functional, not narrative. Pause time is included in the duration. Spoken word budget = `(duration_sec - total_pause_sec) × (85..95 wpm / 60)`.

---

## 3-minute (180 sec)

| Time | Function | Pause budget | Notes |
|---|---|---|---|
| 0:00–0:15 | Orientation: safety, no pressure | 4s | Music starts before first word |
| 0:15–0:45 | Breath + body settling | 5s | Longer exhale framing |
| 0:45–1:30 | Progressive release (face → shoulders → hands → belly → legs) | 6s | Skip muscle-tensing — only softening |
| 1:30–2:30 | Guided imagery + permissive hypnotic suggestions | 7s | One metaphor only |
| 2:30–3:00 | Sleep permission + fade | 6s | Music carries final 15s |

Spoken word target: **170–210 words**.

## Word-Per-Minute Calibration

- Target speaking rate: **85–95 wpm**
- Normal podcast/conversational speech: 140–170 wpm — **never** use this rate
- Calculation example (4-min track):
  - Total: 240 sec
  - Pauses: ~32 sec (rough budget for 4-min track)
  - Spoken time: 208 sec
  - Words at 85 wpm: 295
  - Words at 95 wpm: 329
  - **Practical script length: 220–280 words** (accounting for natural slowing within phrases)

If your script overflows, **cut**, don't speed up. Speed-up kills the parasympathetic effect.

---

## Universal Rules Across All Durations

1. **Music begins before voice** and **continues after voice** — never start or end with voice
2. **Pauses are content** — not gaps to fill
3. **The last 25–30% of every track has zero new ideas** — only repetition and fade
4. **No awakening section** at any duration
5. **Word density is inversely proportional to time** — longer tracks should be more sparse, not more verbose
