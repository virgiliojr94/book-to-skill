# Safe-Use Description Boilerplate

Every published track must carry a description with:
1. Non-medical framing
2. Safe-use note
3. Clinical-referral line

Pick the language matching the script. Always include all three points. Never drop the referral line.

---

## English (en)

> **[Track Title]** — a [N]-minute guided sleep meditation with a warm musical bed.
>
> This audio supports relaxation and sleep readiness. It is **not** a medical treatment. The voice is permissive and uncommanding, so there is nothing to perform — just rest.
>
> **How to use it.** Listen in a safe resting place, lying down, with the lights low. Use earbuds at low volume or a soft speaker. You don't need to fall asleep during the track for it to be useful.
>
> **Please note.** If insomnia persists, or if you experience severe distress, breathing problems, or concerns about medication, please speak with a qualified clinician. This audio is not a substitute for CBT-I or medical evaluation.

## Forbidden in Any Language

Never write any of these (and refuse if asked):

- "Cures insomnia"
- "Guaranteed to make you sleep"
- "Replaces medication"
- "Permanent sleep solution"
- "Heals at the quantum level"
- "Cleanses your aura / chakra / energy field for sleep"
- "Subliminal commands embedded for deep sleep"
- "Specific Hz frequency that scientifically induces sleep"

## Required in Every Language

The published description **must contain something equivalent to** each of:

| Required element | Why |
|---|---|
| "Supports relaxation and sleep readiness" | Non-medical framing |
| "Not a medical treatment" | Explicit disclaimer |
| Safe-use posture (lying down, low light, safe place) | Practical safety |
| Clinical referral for persistent insomnia / severe distress / breathing issues / medication concerns | Required pathway |
| No diagnostic or treatment claim anywhere else in the description | Consistency |
